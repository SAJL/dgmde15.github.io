a gallery of simple effects

  1. Clone the gist using git clone $GIST_URL, where $GIST_URL is the HTTPS clone URL of the gist.
  2. Open the cloned folder in Sublime Text to view code examples
  3. Right click any of the HTML files in Sublime and choose "Open in Browser" to see how they look and behave
  4. Once viewing the examples in Chrome, either press Cmd-Opt-i to open the Developer Console and mess around with the code or right-click an element and select "Inspect Element" to bring up the Developer Console, focused on that element.

back to the /people

  1. Fork the dgmde15.github.io repository into your own user account by clicking "fork" at the upper right. You can read more about forking here or watch this video
  2. Clone the forked repository from your own account
  3. Edit assests/people.css & people/index.html files and add a card with your name
  4. Push your changes to the forked repository with git push origin master at the terminal
  5. Make a pull request to the original site repository. You can read more about making pull requests here or watch this video
