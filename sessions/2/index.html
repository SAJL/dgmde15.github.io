back to the /people page

  1. Fork the dgmde15.github.io repository into your own user account by clicking "fork" at the upper right
  2. Clone the repository from your own account by typing
    git clone https://github.com/$YOUR-USERNAME/dgmde15.github.io
    at the terminal
  3. Edit the assests/people.css and people/index.html files to change the name and photo on your personal card
  4. Push your changes to the forked repository by typing
    git push origin master at the terminal
